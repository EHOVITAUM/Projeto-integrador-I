# Projeto-integrador-I
Projeto integrador desenvolvido durante o primeiro semestre do curso de Big data para negócios na Faculdade de Tecnologia Ipiranga.
O objetivo do projeto é criar  um  banco de dados estruturado fisico(PostgreeSQL), alimenta-lo e posteriormente ger dashboards que possam agragar valor e propor melhoria nos setores de estoque, controle de clientes  e  controle de fornecedores, tudo isso havendo integração de alguma maneira com python(de forma que algumma parte do projeto fosse desenolvido no mesmo). O setor escolhido em questão foi o estoque de uma farmacia hospitalar, o qual populamos com alguns milhares de registros (como tende a ser em uma farmacia hospitalar, na tentativa de ser um projeto fidedigno), e inserimos uma função de CRUD completo atraves de um script em python.
